## Decisões e contexto importantes

- **`config.local.js` como solução de PAT**: Safari apaga `localStorage` para `file://` entre sessões. O arquivo JS local com `window.GH_PAT` resolve sem backend. `onerror="void 0"` garante fallback silencioso no GitHub Pages (onde o arquivo não existe).
- **README de skills auto-gerado**: categorias definidas em `SKILL_CATEGORIES` no `build-site.py`. Ao criar nova skill, verificar se ela cai em "Outras" e adicionar à categoria correta no dict.
- **Aba Decisões usa `sync-targets.yml`**: novos projetos entram automaticamente ao serem adicionados ao YAML — sem alterar o gerenciador.
- **`_template.md` ignorado pelo parser**: o código filtra arquivos que começam com `_` — assim o template não aparece como decisão.
