# Claude Code best practices — Cal Rueb (Anthropic)

**Tema:** metodologia
**Fonte:** https://www.youtube.com/watch?v=gv0WHhKelSE

## Resumo

- Use CLAUDE.md para dar contexto persistente ao Claude sem repetir instruções a cada sessão.
- Divida tarefas grandes em subtarefas menores usando o modo de planejamento antes de executar.
- Skills permitem ensinar o Claude uma vez e reutilizar o comportamento em múltiplos projetos.

## Por que isso importa

Para um Head de Growth sem background técnico, essas práticas são o que separa "conseguir fazer uma coisa" de "fazer de forma consistente e escalável" — o Claude se comporta de forma previsível quando o contexto está bem definido.

## Citação

> "O melhor uso do Claude Code não é pedir para ele escrever código — é ensinar ele a trabalhar do seu jeito."
