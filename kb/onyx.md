# Onyx

**Tema:** ferramentas
**Fonte:** https://github.com/onyx-dot-app/onyx
**GitHub Stars:** 17.3k

## Resumo

- Plataforma de chat com IA auto-hospedada que conecta com 40+ fontes (Google Drive, Slack, Notion, Confluence, GitHub)
- Tudo fica dentro da sua infraestrutura — indicado para empresas com dados sensíveis
- Instalação via Docker: `git clone` + `docker compose up` na pasta `onyx/deployment/docker_compose`

## Quando usar

- Equipe que precisa perguntar sobre documentos internos sem enviar para serviços externos
- "Qual é nossa política de férias?" — Onyx busca no Notion e responde
- Onboarding de novos funcionários com acesso ao repositório de conhecimento
- Suporte técnico interno conectado ao repositório de código e documentação
