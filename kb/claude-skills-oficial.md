# Claude Skills Oficial (Anthropic)

**Tema:** ferramentas
**Fonte:** https://github.com/anthropics/skills

## Resumo

- Repositório oficial da Anthropic com templates de skills para processar PDFs, planilhas Excel, apresentações PowerPoint e documentos Word
- É a forma "correta" e testada pela Anthropic de instruir o Claude a lidar com esses formatos
- Instalação: `git clone https://github.com/anthropics/skills.git` + copiar as skills desejadas para `.claude/skills/`

## Quando usar

- Extrair tabelas de PDF e transformar em dados estruturados
- Ler planilha Excel e gerar análises automáticas
- Processar apresentações e extrair pontos principais
- Criar fluxos de processamento de documentos em lote
