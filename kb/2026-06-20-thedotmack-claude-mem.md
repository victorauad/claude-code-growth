# Claude-Mem: Sistema de Memória Persistente para Claude Code

**Tema:** ferramentas
**Fonte:** https://github.com/thedotmack/claude-mem
**GitHub Stars:** 83371

## Resumo

- Sistema de compressão de memória que preserva contexto entre sessões automaticamente
- Captura observações de ferramentas, gera resumos semânticos e disponibiliza para futuras sessões
- Instalação simples via NPX com suporte para Claude Code, Gemini CLI e OpenCode
- Mantém continuidade de conhecimento sobre projetos mesmo após encerramento de sessões

## Por que isso importa

Para um Head de Growth em Martech, Claude-Mem elimina a necessidade de recontextualizar o IA a cada sessão, permitindo workflows mais produtivos em Claude Code. Isso acelera ciclos de desenvolvimento de features, automações e análises, reduzindo overhead cognitivo e aumentando o ROI de tempo dedicado à IA engineering.

## Citação

> Claude-Mem seamlessly preserves context across sessions by automatically capturing tool usage observations, generating semantic summaries, and making them available to future sessions.
