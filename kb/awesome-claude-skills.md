# Awesome Claude Skills

**Tema:** ferramentas
**Fonte:** https://github.com/travisvn/awesome-claude-skills
**Instalações:** 22k+

## Resumo

- Marketplace comunitário com centenas de skills para Claude Code em áreas como SEO, marketing, segurança, design, finanças e jurídico
- Em vez de escrever prompts longos toda vez, uma skill encapsula o conhecimento especializado
- Instalação: `git clone` + copiar os arquivos SKILL.md desejados para `.claude/skills/`

## Quando usar

- Expandir o Claude Code com conhecimento especializado de domínios específicos
- Skill de SEO: analisa texto e sugere otimizações automaticamente
- Skill de segurança: audita código buscando vulnerabilidades
- Skill jurídica: revisa contratos buscando cláusulas problemáticas
