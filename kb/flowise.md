# Flowise

**Tema:** ferramentas
**Fonte:** https://github.com/FlowiseAI/Flowise
**GitHub Stars:** 30k

## Resumo

- Construtor visual de agentes e chatbots com arrastar e soltar — mais simples que o Dify, focado em prototipagem rápida
- Conecta modelos de linguagem (Claude, GPT) com bases de conhecimento e APIs de forma visual
- Instalação: `npm install -g flowise` + `npx flowise start`

## Quando usar

- Prototipar aplicações de IA em minutos para apresentar a stakeholders
- Criar chatbot que lê PDFs e responde perguntas
- Agente que pesquisa na internet e resume resultados
- Primeiro passo antes de decidir se vale construir algo mais robusto
