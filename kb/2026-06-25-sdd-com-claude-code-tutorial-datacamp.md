# Tutorial — Spec-Driven Development com Claude Code (DataCamp)

**Tema:** metodologia
**Fonte:** https://www.datacamp.com/tutorial/spec-driven-development-with-claude-code

## Resumo

- Tutorial prático do DataCamp especificamente sobre SDD com Claude Code — cobre todo o fluxo do zero
- O CLAUDE.md funciona como memória persistente de specs entre sessões: escrita uma vez, aplicada automaticamente em toda conversa
- Mostra como usar subagentes do Claude Code para execução paralela de tasks derivadas da spec
- Demonstra o uso de hooks para validar automaticamente que o código gerado cumpre a spec antes de commitar
- Inclui template de spec reutilizável: seções de contexto, requisitos funcionais, restrições técnicas e critérios de aceitação

## Por que isso importa

DataCamp é uma plataforma de ensino técnico respeitada, e este tutorial é o mais direto para aplicar SDD com Claude Code especificamente — não requer background de desenvolvedor. Altamente aplicável ao contexto de usar Claude Code como ferramenta de trabalho.

## Citação

> "Claude Code supports spec-driven development across multiple sessions because CLAUDE.md automatically persists instructions — write it once, apply it everywhere."
