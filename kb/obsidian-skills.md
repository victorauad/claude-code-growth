# Obsidian Skills

**Tema:** ferramentas
**Fonte:** https://github.com/kepano/obsidian-skills

## Resumo

- Skills que conectam o Claude Code com o Obsidian — o Claude entende estrutura do vault (pastas, links entre notas, tags, templates)
- Permite que o Claude navegue e manipule seu sistema de conhecimento pessoal como um assistente que conhece suas notas
- Instalação: `git clone` + copiar skills para `.claude/skills/`

## Quando usar

- "Crie uma nota de reunião seguindo meu template padrão"
- Resumir todas as notas com determinada tag
- Encontrar conexões entre notas que você ainda não percebeu
- Mover e organizar notas seguindo a estrutura do vault automaticamente
