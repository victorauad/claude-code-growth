# NotebookLM Integration Skill

**Tema:** ferramentas
**Fonte:** https://github.com/PleasePrompto/notebooklm-skill

## Resumo

- Skill que replica o estilo do NotebookLM (Google): processa coleções de fontes e gera resumos comparativos, flashcards e mapas mentais
- Ideal para absorver grandes volumes de informação rapidamente sem ler cada fonte manualmente
- Instalação: `git clone https://github.com/PleasePrompto/notebooklm-skill.git` + copiar a skill para `.claude/skills/`

## Quando usar

- Processar 10+ artigos e gerar um resumo comparativo
- Criar flashcards automáticos de um capítulo de livro
- Gerar mapa mental de um tema a partir de múltiplas fontes
- Consolidar anotações de reuniões em insights acionáveis
