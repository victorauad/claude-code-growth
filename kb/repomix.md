# Repomix

**Tema:** ferramentas
**Fonte:** https://github.com/yamadashy/repomix
**GitHub Stars:** 20.9k

## Resumo

- Transforma todos os arquivos do projeto em um único texto otimizado para IA, eliminando a necessidade de copiar arquivos um por um
- Reduz tokens mantendo o contexto relevante — ideal para pedidos de refatoração, revisão de arquitetura e onboarding
- Uso imediato sem instalação: `npx repomix` na raiz do projeto gera `repomix-output.txt`

## Quando usar

- Pedir ao Claude para revisar toda a arquitetura do projeto
- Fazer refatorações que impactam múltiplos arquivos
- Gerar documentação automática do codebase
- Onboarding de novos colaboradores
