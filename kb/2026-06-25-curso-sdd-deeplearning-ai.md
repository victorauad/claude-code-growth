# Curso — Spec-Driven Development com Coding Agents (DeepLearning.AI)

**Tema:** metodologia
**Fonte:** https://www.deeplearning.ai/courses/spec-driven-development-with-coding-agents

## Resumo

- Curso oficial da DeepLearning.AI (Andrew Ng) sobre SDD com agentes de código — gratuito e certificado
- Cobre o ciclo completo Specify → Plan → Implement → Validate usando agentes de IA como executor
- Ensina a estruturar specs em markdown que agentes conseguem seguir de forma autônoma por múltiplas sessões
- Inclui módulos sobre como decompor um problema em tasks atômicas para o agente executar passo a passo
- Aborda como o CLAUDE.md funciona como memória persistente de specs entre sessões — relevante diretamente para Claude Code

## Por que isso importa

DeepLearning.AI é a principal plataforma de educação em IA do mundo (fundada por Andrew Ng). Um curso específico sobre SDD com agentes valida que a metodologia é o caminho padrão para trabalhar com coding agents — não uma preferência pessoal.

## Citação

> "Specify requirements in markdown, generate a plan, implement against the plan, and validate results — the four-phase SDD methodology."
