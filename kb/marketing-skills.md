# Marketing Skills

**Tema:** ferramentas
**Fonte:** https://github.com/coreyhaines31/marketingskills

## Resumo

- Suite com 23 skills de marketing para Claude Code: SEO, copywriting, Google Ads, Meta Ads, email marketing, redes sociais e análise de concorrentes
- Cada skill é calibrada para uma tarefa específica, entregando resultados no formato certo sem reescrever prompts
- Instalação: `git clone` + copiar os arquivos SKILL.md para `.claude/skills/`

## Skills principais

- `seo-optimizer` — otimiza textos para mecanismos de busca
- `ad-copywriter` — gera copys para Google Ads, Meta e LinkedIn
- `email-sequence` — cria sequências de email marketing
- `social-calendar` — monta calendário de conteúdo para redes sociais
- `competitor-analysis` — analisa concorrentes e mapeia oportunidades

## Quando usar

- Times de marketing que querem usar IA de forma estruturada e consistente
- Gerar variações de copy para anúncios sem reescrever prompts do zero
- Criar briefing de campanha a partir de um objetivo de negócio
