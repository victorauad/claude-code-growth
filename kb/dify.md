# Dify

**Tema:** ferramentas
**Fonte:** https://github.com/langgenius/dify
**GitHub Stars:** 130k+

## Resumo

- Plataforma visual para construir fluxos de IA arrastando blocos — sem escrever código
- Conecta com fontes de dados (PDFs, bancos, APIs) e publica como aplicação pronta
- Instalação via Docker: `git clone` + `docker compose up` na pasta `dify/docker`

## Quando usar

- Criar chatbots internos que respondem sobre documentos da empresa
- Montar fluxo que recebe PDF e gera resumo automático
- Construir agentes de IA para clientes sem escrever código
- Times sem engenharia que querem usar IA com dados sensíveis no próprio servidor
